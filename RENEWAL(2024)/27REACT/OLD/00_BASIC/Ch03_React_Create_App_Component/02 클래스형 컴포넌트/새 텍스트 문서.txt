클래스형 컴포넌트(Class Component)는 React 컴포넌트를 만드는 데 사용되는 하나의 방법입니다. 이전 버전의 React에서는 클래스형 컴포넌트가 주로 사용되었으며, 최근에는 함수형 컴포넌트와 Hooks가 도입되면서 함수형 컴포넌트가 더 많이 사용되고 있습니다.

클래스형 컴포넌트는 ES6의 클래스 문법을 사용하여 정의됩니다. 클래스형 컴포넌트는 React.Component 클래스를 상속받으며, render() 메서드를 구현하여 UI를 반환합니다.

클래스형 컴포넌트는 다음과 같은 특징이 있습니다.

state와 lifecycle 메서드를 사용할 수 있습니다.
클래스형 컴포넌트는 자체적인 상태(state)를 가질 수 있으며, 이 상태를 변경할 수 있습니다.
lifecycle 메서드를 사용하여 컴포넌트의 수명 주기 동안 특정 시점에 코드를 실행할 수 있습니다.
