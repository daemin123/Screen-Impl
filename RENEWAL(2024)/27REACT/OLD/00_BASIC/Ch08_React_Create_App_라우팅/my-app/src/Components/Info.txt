
React Router는 React 애플리케이션에서 클라이언트 측 라우팅을
 지원하는 라이브러리입니다. react-router-dom은 React Router의
  DOM 바인딩 버전으로, 브라우저에서 사용할 수 있도록 제공됩니다. 
  이를 사용하여 단일 페이지 애플리케이션을 만들 수 있습니다.

아래는 React Router의 주요 구성 요소인 Router, Switch, Route,
 Link에 대한 예제 코드입니다.

Router
Router는 React Router의 핵심 구성 요소로, 브라우저의 URL을 리액트 
컴포넌트와 연결해주는 역할을 합니다. Router 컴포넌트는 단 하나만 
사용해야 하며, 일반적으로 최상위 컴포넌트에서 사용됩니다.

Switch-없어짐 -> Routes로 대체됨..!!!!!!!!!!!!!!!!!!!!!!
Switch는 Route 컴포넌트 중에서 가장 첫 번째로 매칭되는 컴포넌트를 
렌더링해줍니다. 따라서, Switch 컴포넌트를 사용하면 URL에 해당하는
컴포넌트를 렌더링할 수 있습니다.

Route
Route 컴포넌트는 URL과 해당 URL에 매칭되는 컴포넌트를 연결해줍니다. 
path prop은 URL 경로를 나타내며, component prop은 해당 경로에 매칭되는
 컴포넌트를 나타냅니다.
